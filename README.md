# HSL Bills

This is a [Next.js](https://nextjs.org/) app for creating and managing Bills for Healthcare Solutions Ltd.

## Features

- Create, update, and delete Bills.
- View a list of all Bills.
- Filter Bills by date range.
- Sort Bills by date.
- Export Bills to PDF.

## Installation

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.

## Usage

1. Navigate to the root of the repository.
2. Run `npm run dev` to start the development server.
3. Open your browser and navigate to [http://localhost:3000](http://localhost:3000).
4. Use the app to create, update, and delete Bills.
5. View a list of all Bills.
6. Filter Bills by date range.
7. Sort Bills by date.
8. Export Bills to PDF.
